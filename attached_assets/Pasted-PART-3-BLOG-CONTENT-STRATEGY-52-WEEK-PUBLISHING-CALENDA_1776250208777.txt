PART 4: TECHNICAL SEO — URL STRUCTURE AND INTERNAL LINKING
4.1 URL Structure Rules
IMPLEMENT THESE URL RULES ACROSS THE ENTIRE SITE:

Rules:
- All lowercase: /programs/islamic-education NOT /Programs/IslamicEducation
- Hyphens, not underscores: /zakat-calculator NOT /zakat_calculator
- No trailing slashes inconsistency: pick one and 301 redirect all variants to it
- No URL parameters in key pages: /donate NOT /donate?ref=homepage
- Maximum 3 levels deep: /blog/category/article-title

Canonical URLs:
- Every page must have a self-referencing canonical tag
- <link rel="canonical" href="https://azmifoundation.com/this-exact-page-url">
- Set up 301 redirects: www → non-www (or vice versa), enforce HTTPS
4.2 Internal Linking Map
IMPLEMENT THIS INTERNAL LINK STRUCTURE:

Homepage → links to: /about, /programs, /donate, /zakat, /sadaqah, /blog, /volunteer

/donate → links to: /zakat, /sadaqah, /programs/education, /programs/healthcare

/zakat → links to: /donate, /blog/how-to-calculate-zakat, /blog/who-is-eligible-for-zakat, /about

/sadaqah → links to: /donate, /blog/what-is-sadaqah-jariyah, /programs

/programs → links to: /programs/education, /programs/healthcare, /programs/social-welfare, /donate

Every blog post must:
- Include 3-5 internal links to relevant pages/posts
- Include 1 CTA link to /donate or /zakat or /sadaqah
- Include "Related Articles" section at bottom

Use keyword-rich anchor text:
GOOD: "learn about our Islamic education programs"
BAD: "click here" or "read more"

EXPERIENCE signals:
- Add photos and videos from actual programs/events
- Include testimonials from beneficiaries and volunteers (with names and photos)
- Show founder's background and qualifications on About page

EXPERTISE signals:
- Author bios on all blog posts: Islamic scholar or experienced nonprofit professional
- Cite Quran and Hadith references in Islamic content (with proper citation format)
- Link to reputable external sources (Islamic institutions, academic research)
- Show any affiliations with Islamic organizations or universities

AUTHORITATIVENESS signals:

- Register on Charity Navigator and display the badge
- Show 501(c)(3) status clearly on every page (footer)
- Display annual impact report prominently

TRUSTWORTHINESS signals:
- Privacy Policy page (required for GDPR/CCPA and Google trust)
- Secure HTTPS on entire site
- Clear financial transparency (where does the money go?)
- Physical address in footer
- Phone number and email visible
- Social media profiles linked and active
- No broken links anywhere on the site
- Fast-loading, professional-looking site
